## Install

1. Download **[Jot-2.7.0.dmg](https://github.com/parthha12/jot-app/releases/latest)** from Releases.
2. Drag **Jot** to Applications.
3. If macOS blocks the app:

```bash
xattr -cr "/Applications/Jot.app"
```

4. Allow **Screen Recording** and **Automation** when prompted.
5. Paste your **Anthropic API key** on the Notes tab to enable Taquavion (optional).

Full release notes: **[v2.7.0](https://github.com/parthha12/jot-app/releases/tag/v2.7.0)**

## What you get

- **⌘N** capture · **⌘P** search · **⌘⇧R** recall · **⌘⇧P** Taquavion
- Time and app-open reminders · screen OCR search · saved screen sessions (⌘⇧3)
- One calm overlay · local data on your Mac

This repository distributes the **Mac app only**. It does not contain application source code.
